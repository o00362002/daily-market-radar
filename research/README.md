# Research

本資料夾用於保存每日市場情報系統的方法論研究、公開參考、工具比較與未來改版研究。

研究資料不是正式規則。正式採納前，需先整理成設計決策，再同步到 `SYSTEM_PROMPT.md`、`configs/`、`templates/` 或 `CURRENT_DECISIONS.md`。

---

## 建議結構

```text
research/
├─ README.md
├─ 00_EXECUTIVE_SUMMARY.md
├─ 01_FULL_RESEARCH_REPORT.md
├─ 02_PUBLIC_REFERENCES.md
├─ 03_COMPARISON.md
├─ 04_DESIGN_DECISIONS.md
├─ 05_FUTURE_RESEARCH.md
└─ diagrams/
```

---

## 適用內容

- 市場情報方法論研究
- 公開雷達系統或研究流程比較
- AI 搜尋與來源驗證方法
- 報告格式改版研究
- 漏抓原因與搜尋策略研究
- 指標與資料來源擴充研究
