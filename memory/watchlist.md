# Watchlist｜每日固定追蹤清單

本檔保存使用者長期關注、容易漏抓、或需要跨日觀察的題目。

---

## 1. 加密潛力市場

每日固定掃描：

- RWA
- tokenized stocks / pre-IPO
- OpenAI / SpaceX / Anthropic pre-IPO 相關產品
- AI agents x crypto
- x402 / agent payments
- Perp DEX
- Solana / Monad / Base / Sui / Hyperliquid
- privacy coins / ZK privacy
- ETF / OI / TVL / Fees / Revenue
- stablecoin supply / exchange flows

判斷重點：

- 是否有資金流明顯轉向？
- 是否形成商業閉環？
- 是否只是敘事熱度，沒有用戶、收入或流動性？

## 2. 零售、通路、商圈與品牌變化

每日固定掃描：

- 百貨 / 購物中心
- 街邊店
- 高街與商圈洗牌
- 展店 / 收店 / 撤櫃
- tenant mix
- 體驗型門市
- 台灣商場開幕與國際品牌動向
- 零售人流與消費壓力
- 中價品牌壓力與低價零售份額

判斷重點：

- 對實體零售轉換有沒有影響？
- 對商品企劃、會員經營、門市體驗有沒有影響？
- 是否是單店事件，還是通路結構變化？

## 3. AI 實際應用與工作流替代

每日固定掃描：

- AI agent 客服
- 權限治理
- 身份管理
- MCP / 工具層
- 零售 / 客服 / BI / 營運 / 內容工作流
- AI 影片與內容生產
- 企業導入與 ROI
- 就業、招聘、外包價格變化

## 4. 科技發展過熱指標

每日固定掃描：

- AI 收入與企業 ROI
- Cloud AI CapEx
- GPU / HBM / 電力
- NVIDIA 訂單與供應鏈
- 資料中心債務
- 私募估值
- 裁員 / 招聘

## 5. 社群內容與真分眾 / 假分眾

每日固定掃描：

- Threads
- Instagram
- TikTok
- YouTube Shorts
- 小紅書
- creator monetization
- social search
- platform algorithm
- AI-generated content saturation
- authenticity signals

- UGC / KOC / brand community
