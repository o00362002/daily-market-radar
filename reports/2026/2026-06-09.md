## 9. AI 與實際應用

### 原播報已覆蓋

- AI PC / local inference。
- OpenAI Sites / Codex 類內部工具生成。
- Physical AI / Nvidia Cosmos 類方向。

### 播報後確認的漏抓

#### OpenAI Codex / Enterprise 遷移與兩個月免費訊號

- 狀態：部分確認。官方文件確認 Enterprise / Codex seat、token-based usage、workspace agents、Sites 等企業功能；「兩個月免費切換」當時未找到完整官方頁，需標示官方待補驗證。
- 意義：AI 競爭從模型能力進入企業遷移、工作流鎖定、用量補貼。

#### Anthropic 高 token / rate limit 提升

- 狀態：確認。Anthropic 官方 2026/05/06 宣布提高 Claude Code 與 Claude Opus API rate limits，與 SpaceX Colossus 1 compute partnership 有關。
- 意義：compute capacity 成為產品競爭武器；agent 長任務能不能跑，取決於 token、context、工具調用與重試成本。

#### OpenAI Dreaming / Memory

- 狀態：確認。OpenAI 2026/06/04 發布 Dreaming / memory 更新，目標是改善 ChatGPT 長期記憶、偏好、ongoing work 的上下文。
- 意義：長期記憶與個人化上下文成為 agent 與個人工作流地基。

#### Anthropic Dreams / Dreaming

- 狀態：確認。Anthropic Dreams / Dreaming 屬 Claude Managed Agents research preview，會讀取過去 sessions 與 memory store，產生新的整理後 memory store。
- 意義：這不是模型權重自我訓練，而是 agent 把工作日誌整理成 SOP、playbook、錯誤清單與未來任務記憶。
