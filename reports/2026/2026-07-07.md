## 2. 今日核心訊號

### A. AI / Agent / 工作流

#### AI-1｜英國 FCA 要求評估是否監管通用 AI 模型

- 事件：英國金融監管單位關注 ChatGPT、Claude、Gemini 等通用 AI 對金融建議與消費者決策的影響。
- 判斷：AI 監管從企業內部工具走向顧客決策入口。
- 證據等級：high
- 台灣映射：未來金融、保險、投顧、客服與理財平台導入 AI 時，責任邊界會更重要。

#### AI-2｜Codex / agentic AI 使用行為出現工作流遷移證據

- 事件：研究指出 Codex active users 快速成長，部分組織內部使用從 ChatGPT 轉向 Codex / agentic workflow。
- 判斷：AI 使用重心從聊天介面轉向任務型 agent。
- 證據等級：medium-high
- 與使用者專案關聯：RetailOps Agent System 應優先設計 task routing、tool use、output validation，而非只做聊天介面。

#### AI-3｜Microsoft 早期 rollout 研究：CLI coding agents 與 PR 產出正相關

- 事件：研究 Microsoft 早期 2026 rollout 的論文指出 CLI coding agent 使用者 merged PR 數增加，但 merged PR 不等於商業價值。
- 判斷：Agent ROI 需要用 workflow metric 衡量，而不是只看模型能力。
- 證據等級：medium-high

#### AI-C1｜候選：AI orchestration 成為企業 AI 成本控制層

- 事件：Microsoft 將 Copilot / Foundry 定位為 model-agnostic orchestration layer。
- 判斷：企業未來可能不是買單一模型，而是買任務路由、成本控管與治理層。
- 證據等級：medium

#### TW-AI-1｜台灣 AI server 出口管制調查延伸至供應鏈治理

- 事件：Super Micro 台灣單位員工因 AI server 涉嫌非法出口中國的調查遭拘留或交保。
- 判斷：AI 供應鏈治理包含文件、客戶、終端流向與出口合規。
- 證據等級：high
- 分類修正：此事件應放 AI / AI supply chain，不得拿來填科技發展 quota。
