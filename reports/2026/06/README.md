## 後續固定規則

1. 每日報告必須保留來源位置與事件時間。即使簡短，也不能無來源帶過。
2. 重大事件必須先評估重要性、可信度、熱度，不能因媒體重複轉載就自動提高重要性。
3. S 級事件不得一句帶過，必須拆：事件、數字、原因、外溢、風險、下一步。
4. 真分眾 / 假分眾不得給沒有方法支撐的假精準百分比。
5. 若資料不足，必須標示資料缺口與下一步驗證。
6. 使用者指出的漏抓需進入 `memory/missed_cases.md` 或報告末段回測面板。
